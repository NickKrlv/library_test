# Библиотека на Python

Этот проект представляет собой простую библиотечную систему, написанную на Python. Она позволяет добавлять, удалять, искать и просматривать книги, а также управлять их статусом (в наличии/выдана).

## Функциональность

* **Добавление книги:** Добавляет новую книгу в библиотеку с указанием названия, автора и года издания.  Автоматически генерируется уникальный ID для каждой книги.
* **Удаление книги:** Удаляет книгу из библиотеки по ее ID.
* **Поиск книги:** Позволяет искать книги по названию, автору или году издания.
* **Просмотр всех книг:** Выводит список всех книг, хранящихся в библиотеке, с указанием их ID, названия, автора, года издания и статуса.
* **Изменение статуса книги:** Позволяет изменить статус книги с "в наличии" на "выдана" и наоборот.
* **Логирование:** Все операции (удаление, изменение статуса) логируются в файл `library.log`.
* **Сохранение данных:** Данные о книгах хранятся в JSON-файле `library.json`, что обеспечивает их сохранение между сеансами работы программы.
* **Цветной вывод в консоль:** Для удобства использования, вывод в консоль оформлен с помощью цветовых выделений.

## Установка и запуск

1. Убедитесь, что у вас установлен Python 3.
2. Скопируйте код из файла `library.py` в ваш локальный файл.
3. Запустите программу из командной строки: `python library.py`

## Использование

После запуска программы вам будет представлено меню с доступными действиями.  Следуйте инструкциям на экране для выполнения нужной операции.

## Структура проекта

* `library.py`: Основной файл с кодом библиотеки.
* `library.json`: Файл для хранения данных о книгах (создается автоматически).
* `library.log`: Файл для хранения логов (создается автоматически).

## Зависимости

Проект использует следующие библиотеки:

* `json`: для работы с JSON-файлами.
* `logging`: для логирования событий.
* `os`: для работы с файловой системой.
* `datetime`: для работы с датами и временем.
* `typing`: для аннотации типов (необязательно, но рекомендуется для повышения читаемости кода).
